HTML

Una de las herramienats más destacadas en sin lugar a duda es la programación de páginas web con HTML, podemos hacer infinidad de cosas con esto, como desarrollar nuestras hábilidades en la inscripción de scripts para mejorar la vista de un menú o por ende insertar textos en una página, logrando así avanzar en nuestros conocimientos de la matería.

  • Creación de páginas web con la utilización de sublime text
  • Utilizamos comandos básicos para html
  • Diferenciamos párrafos brincando renglones
  • Dimos color al fondo de la pantalla
  • Creamos un menú principal con hipervinculos
  • Añadimos etiquetas para hacer varias combinaciones en la página
  • Así también como la utilización de imágenes como hipervinculos

CSS

Pero para todo texto debe haber fuentes y colores para poder distinguirlo de entre los demás, de esto se encarga el css, por lo que si queremos hacer que un texto sea visualmente atractivo para los usuarios requerimos de esta herramienta para poder lograrlo

Utilizamos las siguientes caracteristicas:

  • Creamos fuentes fáciles de añadir a los párrafos de texto en las páginas
  • Pudimos hacer enfasis en la utilización de formatos como la generación o eliminación de simbolos
  • Hicimos que fuera fácil de visualizar y atractivo para el público en general
  • Diferenciar cada parte del texto con esta herramienta

Opinión personal

Es debido a estas herramientas que podemos observar lo que se muestra en todas las páginas anteriores y también con la gran diversidad de etiquetas que se poseen y se pueden utilizar para darle la forma que nosotros queramos, haciendo fácil y cómodo de trabajar desde la comodidad de tu casa.
